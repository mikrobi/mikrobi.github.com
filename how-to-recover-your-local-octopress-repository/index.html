Jan 30th, 2016

I am hosting my blog on GitHub pages. Due to a broken hard disk I’ve recently lost my local respository containing the Octopress sources. A simple git clone is not enough to recover the local repository because Octopress works with two branches: source for keeping the files to generate the static site and master that contains the site itself.
